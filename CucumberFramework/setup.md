Installation 

BDD - Cuucmber ----> plain english testcases (.feature file) ---> Step def (logic - code for that particular line)
1. npm init -y
2. npm init playwright@latest
3. npm install -D @cucumber/cucumber
4. npm install -D typescript ts-node
5. npm i dotenv

To generate Step def code block - and to run
npx cucumber-js --config .\config\cucumber.json
